# Examples

## Canonical public beta runner

- `evolve_language.rs`

This is the only canonical public beta run path in the current tranche. It is the runner that should be used in public docs, CI smoke runs, and release notes.

## Experimental research examples

All other examples in this directory are retained for research continuity:

- sweeps
- breed and crystallize lines
- pocket and depth probes
- topology and init probes
- performance diagnostics and benches

They are useful, but they are not the public beta compatibility promise.
