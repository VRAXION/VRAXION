ASCII mental model

Lane glossary

Highway / overlap laneInput bit közvetlenül rá van kötve egy A-lane-re, és ugyanaz az edge visszafelé is dekódol. Ez stabil, de copy-szerű.
Mirror / redundant laneUgyanaz a bit második A-lane-en is megjelenik. Ez adja a baseline robustságot és edge-drop túlélést.
Geometry overlayKis súlyú extra edge-ek, amelyek az A16 távolságokat formálják. Ezek javíthatják az ASCII-geometriát, de vékonyíthatják a margint.
Hidden-only / no-prefill laneNincs előre bekötött highway. A keresés maga talál edge-eket. Naturalabb lehet, de jelenleg gyenge marginú.

Mutation model in this playground

D21F search mutates the encoder E: add edge bit_j -> A_i remove edge delete bit_j -> A_i flip edge move endpoint to another bit/lane reweight edge change weight in {-1, -0.75, -0.5, -0.25, +0.25, +0.5, +0.75, +1} Decoder is not separately mutated: decode uses E.T So every visible line is reciprocal: bit_j <-> A_i with same weight
