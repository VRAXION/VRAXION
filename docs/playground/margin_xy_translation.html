Margin Gate — XY Fordítás

3 tanult szám az X-en (a, b, c) → 1 kimenet az Y-on

a +3.0
b +7.0
c -5.0
A lépések
  1. 3 tanult szám: a=+3.0, b=+7.0, c=-5.0
  2. Magnitúdó szerint sort: 3 < 5 < 7
  3. CUT a középsőnél: Y = 5 (|c|)
  4. Legnagyobb fölénye: |b| - |c| = 7 - 5 = 2
  5. Előjel legnagyobbtól: b=+7 → kimenet +2.0
