02 model scope

Local-first, not a hosted model wrapper.

The planned T1 artifact is scoped to avoid hosted model-wrapper dependencies. Final dependency disclosure belongs with the signed artifact, so judge this preview by release evidence and bounded behavior rather than broad AI claims.

Owned path

The public target is framed as a VRAXION runtime path, not someone else's remote answer stream with new paint.

Exact mode first

Supported paths answer, unsupported paths refuse, and speculative output must be visibly labeled before release.

Release-gated growth

Capability should grow through scoped mechanisms, tests, and public artifacts, not through broader claims ahead of evidence.

mode contract

Exact mode refuses unsupported paths.

The release target is simple to judge: supported paths answer, unsupported paths refuse, and creative drift stays opt-in, bounded, and traceable.

Refuse invention
