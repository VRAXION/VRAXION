02 position

Not AI. Not ever.

There is no neural net inside the public promise. INSTNCT should be judged as a local reflex engine with bounded behavior, not as another hosted model wrapper with new paint.

Own fabric

The α-Sync substrate is a VRAXION surface, not someone else's remote model dependency.

Structural, not probabilistic

Exact mode belongs to bounded paths and refusal behavior, not sampled invention hidden inside default answers.

Scales by dimension

Capability grows through internal structure and curated skills, not through bigger neural weights.

flagship capability

Hallucination, toggleable.

Two modes. Exact recall only emits supported transformations and refuses when support is missing. Imagination can exist, but it must be visibly labeled as creative speculation.

Planned for the free T1 release target.
mode contract exact by default

behavior

Refuse invention

Only seen-and-approved pattern transformations are emitted. No approved path means the fabric returns a refusal instead of a fabricated answer.

query -> selector -> approved path -> bounded output no approved path -> "i don't have this"
behavior

Bounded drift

Imagination is opt-in. The fabric may compose beyond seen patterns, but the output is visibly creative and keeps an inspectable trace.

query -> selector -> composed path -> bounded output drift budget enforced, trace stays inspectable

09 model note / T1
