Architecture line

INSTNCT

INSTNCT is Vraxion's current architecture line: a gradient-free self-wiring system that changes its own directed graph instead of learning inside a fixed layer stack.

The public architecture claim is narrow and explicit: fixed I/O surfaces, a sparse signed recurrent graph, persistent state across ticks, and mutation-selection training.

What makes INSTNCT different

The learnable object is the hidden graph itself, not only weights inside a frozen topology.

Self-wiring graph

Topology is part of the search space. Evolution changes sparse routing instead of only tuning dense internal weights.

Persistent runtime state

Charge and state survive across ticks, so computation unfolds over time instead of collapsing into a single pass.

Compact neuron controls

Controls such as theta, channel, and polarity shape selectivity and timing without exploding the search space.

Explicit public boundary

Shipped defaults, validated alternatives, and still-experimental lines remain distinct in the public story.

Fixed vs learnable

The architecture rule stays simple: keep the recurrent substrate learnable, and keep the public boundary explicit.

Component Status Why it matters
Input interface Fixed Public boundary stays readable and comparable across experiments.
Output interface Fixed Readout remains explicit instead of dissolving into the hidden graph.
Hidden graph / mask Learnable Core object that mutation-selection changes.
theta, channel, polarity Learnable Compact neuron roles change timing, thresholding, and sign behavior.
Charge / state Runtime state Computation persists across ticks rather than acting like a one-pass feedforward cascade.
