Positioning / Not AI

Not AI. Not ever. A bounded runtime surface.

The GLM redesign framed the public promise more sharply: INSTNCT should be evaluated as a local reflex engine, not as another neural chatbot wrapper. The public page now says that directly.

No neural net posture

No weights, no remote model dependency, no frontier claim.

Public language stays narrow: T1 is a local-runnable baseline target for bounded, inspectable behavior. It should earn trust through repeatable checks, not branding language.

Local-first

Pull the network and the premise still has to stand.

The intended verification path is direct: download, run locally, compare behavior, and measure the hot path on your own machine.

Own fabric

VRAXION infrastructure, not someone else's model with new paint.

Prismion and alpha-Sync remain public-facing names for the data-refraction substrate. The implementation details stay bounded until release artifacts can be tested.

Hallucination mode

Hallucination becomes a switch, not a hidden failure mode.

The zip version treated exact recall and imagination as explicit modes. This static public page keeps the same product boundary: exact mode should refuse invention; imagination mode should be labeled as creative speculation.

Exact recall Default

Bounded answers, refusal when support is missing, traceable result path.

Imagination Opt-in

Creative generation can exist, but it must be visibly marked as imagination.

Model note / T1 - Tier 1 Reflex Class

Tier 1: a fast local baseline for bounded reasoning.

T1 means Tier 1 in the INSTNCT capability ladder: a free-to-download target for local, low-power, noncommercial experimentation before higher creative and long-horizon tiers arrive.

Latency target

Designed around microsecond-class hot paths.

The goal is minimal reaction delay: hot runtime work should stay in the microsecond range where the device, build, loaded surface, and task shape allow it. Current public wording treats this as a measured direction, not an end-to-end latency guarantee.

Prismion basis

Reasoning built from data refraction units.

The public explanation stays high-level: Prismion comes from prism plus neuron. Instead of light, a Prismion refracts data into the α-Sync fabric, where propagation and interaction produce a bounded result path.

Operating substrate

α-Sync fabric turns interaction into a checkable path.

The α-Sync fabric is the substrate the data is projected into. The intended result is deterministic enough to inspect and test, while leaving room for future opt-in test-time learning research on local user skills.

Boundary Fast baseline, not an all-knowing system

Public questions / Release boundary

The skeptical read should still understand the claim.

The zip added a fuller FAQ. This static release keeps the parts that matter for a public GitHub Pages surface: what INSTNCT is, what it is not, and what must be verified later.

Is INSTNCT an LLM wrapper?

No. The public positioning is deliberately "not AI" in the common hosted-neural-model sense. Current source is source-available, and the release story is local verification.

What does symbol grounding mean here?

Publicly, it means results should stay tied to bounded anchors and checkable paths. Internal mechanism details stay limited until release artifacts make the claims testable.

Can it hallucinate?

Exact mode should refuse unsupported answers. Creative or speculative output belongs in an explicit imagination mode, not hidden inside the default answer path.

When can people run it?

T1 is still a release target, not a finished public binary claim. The next honest milestone is a local artifact with benchmark and refusal behavior people can reproduce.

Signal T1 is coming. The first useful version has to be small, local, and measurable.
