02 engine scope

A local reasoning engine, not a hosted answer stream.

The T1 Reflex Engine is the named public target: local path selection, bounded output, and Exact Mode refusal before any broader AI claim. Public now: preview, SDK/docs tag, status docs, capability notes, and scoped benchmark notes. Not public yet: signed T1 binary, checksums, reproduction harness, raw timing logs, or refusal examples.

INSTNCT engine line

INSTNCT names the first VRAXION reasoning engine surface instead of hiding the product behind generic AI language.

Exact Mode first

Supported paths answer, unsupported paths refuse, and speculative output must be visibly labeled before release.

Proof Pack before claims

Capability should grow through scoped mechanisms, signed artifacts, checks, and public notes, not through broad claims ahead of evidence.

mode contract

Exact Mode refuses unsupported paths.

The engine contract is simple to judge: supported paths answer, unsupported paths refuse, and creative drift stays opt-in, bounded, and traceable.

behavior

Refuse invention

Only seen-and-approved pattern transformations are emitted. No approved path means the engine returns a refusal instead of a fabricated answer.

Bounded drift

Imagination is opt-in. The engine may compose beyond seen patterns, but the output is visibly creative and keeps an inspectable trace.
