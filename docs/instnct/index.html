02 position

Not AI. Not ever.

Own fabric

questions

Things people rightly ask.

Skepticism is the correct response. These are the honest answers the public page can give now.

Is INSTNCT an LLM wrapper?

No. The public positioning is deliberately not AI in the hosted-neural-model sense. The release story is local verification.

What does symbol grounding mean here?

Publicly, it means results should stay tied to bounded anchors and checkable paths. Internal mechanism details stay limited until release artifacts make the claims testable.

Can it hallucinate?

Exact mode should refuse unsupported answers. Creative or speculative output belongs in an explicit imagination mode, not hidden inside default answers.

When can people run it?

T1 is still a release target, not a finished public binary claim. The next honest milestone is a local artifact with benchmark and refusal behavior people can reproduce.
