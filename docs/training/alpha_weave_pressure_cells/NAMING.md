# Naming Convention: α-Sync

Canonical display name:

```text
α-Sync
```

ASCII fallback for code and file names:

```text
alpha_sync
```

Do not use `a-Sync` as the canonical name. Use `α-Sync` when Unicode is allowed.

## Meaning

An `α-Sync` Operator translates an outside surface form into the internal canonical form that the Operator system understands.

```text
external wording / glyph / surface form
-> internal canonical Operator language
```

Examples:

```text
× -> *
÷ -> /
− -> -
visible alias -> canonical lexeme
visible unit/code -> canonical unit value
visible claim text -> canonical evidence proposal
```

## Placement

Use `α-Sync` for Operators that perform grounding, normalization, alias binding, glyph conversion, unit/code normalization, or surface-form to canonical-form synchronization.
