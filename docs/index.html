What VRAXION Is

VRAXION is a company and engineering program focused on a new architecture line: passive I/O, persistent internal state, and mutation-selection training in a self-wiring graph.

Why It Matters

The learnable object is a directed ternary graph, not a dense weight stack. The aim is to scale internal structure and persistent dynamics while keeping the reference implementation inspectable and reproducible.

Current Mainline

The only settings that count as live mainline defaults are the ones actually shipped in v4.2/model/graph.py. If an issue or wiki page says more than the code does, the code wins until promotion happens.

Validated Findings

  • flip beat float weight perturbation on English training.
  • scale=1.0 + low theta beat the older scale=3.0 hack.
  • Both are tracked as evidence, not yet as shipped defaults.

Status Taxonomy

  • Current mainline: what is actually shipped in code on main.
  • Validated finding: an experimentally supported result that has not been promoted into the canonical code path yet.
  • Experimental branch: an active build target or design direction, not a live default.
