VRAXION / Mainline Cleanup

Self-Wiring Graph Only.

The default branch is now scoped to the active self-wiring graph line. Historical VRAXION codepaths were removed from the main tree so the repo, the docs, and CI all describe the same thing: the v4.2 self-wiring graph stack.

Core

v4.2/model/graph.py is the stable NumPy reference. It couples sparse signed edges, persistent charge, and gradient-free mutation/selection training.

Training

v4.2/lib/utils.py contains the generic score helpers plus cyclic rewire -> crystallize training used in the stronger self-wiring runs.

Benchmarks

  • tests/test_model.py for adversarial stress
  • tests/rng_tier_benchmark.py for RNG sensitivity
  • tests/sparse_scaling_benchmark.py for scaling probes

Out Of Main

The unfinished surprise learning branch is intentionally excluded from the cleaned mainline until it beats the mutation baseline and its learning rule is redesigned.
