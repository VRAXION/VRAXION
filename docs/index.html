What Vraxion is

Vraxion is both a research program and an engineering program organized around one architecture line. The public ambition is large, but the public standard is stricter: theory, implementation, chronology, and proof stay separate on purpose.

That separation is not caution theatre. It is how the project keeps itself legible while pushing toward machine consciousness as an engineering reality rather than a slogan.
