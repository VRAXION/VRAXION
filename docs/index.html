What VRAXION is

The engine contract: answer on-path, refuse off-path.

This is the VRAXION AI/reasoning surface in public: not a chatbot identity, but a named engine with modes, paths, and verification gates.

01 / scope

Path Selector.

The engine target starts by selecting a known path. If no supported path exists, the answer should not be forced.

02 / gate

Exact Mode.

Exact Mode is the default contract: answer only inside supported paths, otherwise refuse cleanly.

03 / audit

Proof Pack.

The runnable claim should arrive as a signed artifact, checksums, notes, and local verification steps.
