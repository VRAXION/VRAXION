phi_destinations
→ Tensor
Builds the jump destination lookup table for all experts. Uses golden ratio (1/φ ≈ 0.618) to space jumps maximally apart on the ring. dest[i, j] = (j + floor(M/φ) + i) % M. Called once in __init__, stored as buffer — never recomputed.
(n_int, m_int) → (N, M)

func_softwrit_tns
→ Tensor
Differentiable write back into the ring. Broadcasts hidden state to window → scatter_add at window slots. Additive — accumulates, never erases. No write projection in v4 — writes raw hidden state directly.
(ring_tns, hidden_tns, expanded_idx_tns, weights_tns) → (B, M, D)

func_movepntr_tns
→ Tensor
Moves one expert's pointer. Two options soft-blended: φ-jump (golden-ratio destination) or +1 walk (short step). p·jump + (1−p)·walk — fully differentiable, no hard switching. Pointer stays fractional so gradients flow through.
(ptr_tns, dests_tns, prob_flt, slots_int) → (B,)

Context read scale. Fixed 0.05. Scales how much projected ring reads blend into hidden state. Swept 0.00-0.50 across 6 network sizes (CPU+GPU): 0.05 wins 4/6, best geo mean. Larger nets prefer slightly higher (xlarge optimal ~0.15). Requires read_proj Linear(D,D) on the read path.
Jₑ
CNFG_JUMPIPROB_FLT
float
Explorer jump probability. Fixed 0.9. Stride gate blend weight for explorer-role beings. Swept 0.5–0.95 across 6 sizes (CPU+GPU), 10 seeds: loss surface flat — seed variance 14× param effect. 0.9/0.1 = geo mean #1 (CPU+GPU combined). Was nn.Linear(D,1) — eliminated.

CNFG_WALKIPROB_FLT
float
Walker jump probability. Fixed 0.1. Stride gate blend weight for walker-role beings. Complement of explorer: walkers prefer sequential pos+1 steps. Neutral beings use fixed 0.5.
