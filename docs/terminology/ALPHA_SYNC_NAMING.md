# α-Sync Naming

Canonical display name:

```text
α-Sync
```

ASCII fallback for code and file names:

```text
alpha_sync
```

Use `α-Sync` when Unicode is allowed. Use `alpha_sync` for code, file names,
or systems that require ASCII.

## Meaning

An α-Sync component synchronizes an observed external surface form with the
internal canonical representation used by the Operator runtime.

```text
external surface form
-> canonical internal representation
-> proposal checked by Agency/governance
```

Examples include glyph normalization, alias binding, unit/code normalization,
and visible surface-form grounding into a bounded proposal.
