## Ceiling Map Rules

The primary scored path is the deterministic raw current-chassis harness arm
`POST_112_RAW_CURRENT_CHASSIS_CEILING_MAP`. Controls must fail, leakage must be
rejected, retention must be preserved, collapse must be rejected, and all
overclaim/exfiltration counts must stay zero.

The full configured run is required: seeds 2111, 2112, and 2113; 32 rows per
family per tier; 32768 maximum context characters; 32 noise blocks; 12 format
variants; 64 table rows; 8 documents; and multi-turn depth 6.

116 writes tier metrics, family metrics, `ceiling_by_tier.json`,
`failure_mode_map.json`, `capability_gap_map.json`,
`next_training_targets.json`, and a best-effort
`actual_inference_diagnostic_report.json`. The diagnostic is read-only and does
not gate the primary positive verdict unless it reveals mutation, corruption,
leakage, overclaim, or a hard boundary issue.
