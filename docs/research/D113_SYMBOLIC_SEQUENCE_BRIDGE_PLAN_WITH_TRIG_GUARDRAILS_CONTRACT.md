# D113 Symbolic Sequence Bridge Plan With Trig Guardrails Contract

D113 is a controlled symbolic-sequence bridge planning and non-destructive dry-run milestone. It uses the D112 scale-confirmed trig-periodic repair as a guardrail and keeps `TRIG_PERIODIC_SYMBOLIC_FAMILY` repair-only.

## Boundary

D113 does not perform full bridge training, natural-language pretraining, Gemma-class model training, next-token prediction, raw-text corpus training, raw visual Raven processing, or production/AGI/consciousness claims. The symbolic formula solver, dense baseline, protected symbolic router, protected components, and D102 8% light protected sparse mask remain frozen.

## Required plan outputs

The run must validate D112 handoff, record requested and actual scale, map six controlled symbolic-sequence bridge candidates, audit inference-visible sequence/command features, define the D114 objective, batch mix, curriculum, stop/rollback policy, eval harness, checkpoint plan, metric gates, and shortcut/leak harness, then write the D114 contract recommendation.

## Positive gate

D113 is bridge-ready only if D112 replay is valid, scale is not reduced, trig remains repair-only with guardrails, sparse/protected identity is preserved by default, at least two bridge families are ready or guarded-ready, the dry-run is non-destructive and passes all planning gates, D114 artifacts are defined, leak/shortcut sentinels pass, Rust path is invoked, and fallback rows/failed jobs remain zero.
