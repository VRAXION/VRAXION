```text

```

```text
docs/research/OPERATOR_LIBRARY_CARDS.md
```

Use this file only as a compatibility pointer for older notes and links that
still say `Pocket Library`. New documents should use:

```text

```

term used at the time of those runs. Do not rewrite old evidence claims just to
match the new naming.
