# D44 Formula Primitive Discovery Plan

Status: planning only. This is not a D44 result.

## Goal

D44 should remove the fixed formula-candidate primitive values used by D42/D43 and test whether formula primitives can be learned or discovered from raw symbolic support boards.

The target pipeline is:

raw symbolic support boards -> learned formula primitive candidates -> learned support-evidence extractor -> rule-family inference -> confirmed router path -> selected pocket.

## Starting Boundary

D43 scale-confirmed learned raw symbolic support-evidence extraction only when fixed formula primitive candidates were available. D43 does not prove formula primitive discovery, raw visual Raven reasoning, full hidden-rule Raven solving, natural-language reasoning, DNA/genome success, Raven solved, architecture superiority, consciousness, or general intelligence.

## D44 Requirements To Define

- remove direct fixed candidate formula values from the learned arm inputs
- keep oracle/fixed-primitive upper-bound arms for comparison
- preserve unique target-pocket invariants and unambiguous support evidence
- add controls for primitive-channel shuffling, no support cells, no center cells, and query-only shortcuts
- add audits proving no precomputed support-evidence vector, no boolean equality leak, no formula-value oracle in learned primitive-discovery arms, no target symbol oracle, and no true family leak
- retain same-query-different-support and wrong-support causal controls

## Expected Outcomes

D44 should be treated as a new prototype milestone. A positive result would be bounded to controlled symbolic formula primitive discovery and would not imply raw visual Raven reasoning or broad hidden-rule solving.
