# INSTNCT Private Pilot Issue Triage Policy

Status: 065 private pilot issue triage policy.

Issue triage is for local/private evaluation readiness. It is not production
support and does not create an SLA.

## Severity Levels

Critical:

- suspected sensitive data exposure
- regulated use request
- production/public-beta misuse
- checkpoint/artifact integrity failure
- rollback failure
- policy guard bypass

High:

- service alpha healthcheck failure
- security/supply-chain checker failure
- ops-readiness checker failure
- repeated smoke failure without workaround

Medium:

- onboarding confusion
- documentation mismatch
- non-blocking visual or report issue
- support handoff delay

Low:

- typo
- formatting issue
- non-blocking wording improvement
- optional usability note

## Required Critical Response

- stop the candidate run.
- preserve artifacts.
- notify incident contact.
- use rollback/disable path.
- record issue in post-pilot report.
- require go/no-go re-evaluation.
