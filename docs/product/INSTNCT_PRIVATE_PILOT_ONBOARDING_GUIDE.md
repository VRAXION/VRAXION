# INSTNCT Private Pilot Onboarding Guide

Status: 065 private pilot onboarding guide.

This guide describes readiness steps for a candidate review. It does not launch
a pilot or authorize external use.

## Onboarding Sequence

1. Confirm the candidate is in non-regulated evaluation scope.
2. Review 059 pilot boundary and rejection reasons.
3. Review 060 license package drafts and counsel-review gate.
4. Review 061 RC_001 package and smoke guide.
5. Review 062 service/API alpha boundary.
6. Review 063 security/supply-chain source gate.
7. Review 064 ops readiness gate.
8. Complete data/ops handoff checklist.
9. Assign support owner, support channel, incident contact, and rollback/disable owner.
10. Apply binary go/no-go gate.

## Candidate Instructions

- Use local/private evaluation only.
- Use synthetic, public non-sensitive, or separately approved data only.
- Keep all outputs in the approved local/private path.
- Preserve progress logs, audit logs, summaries, reports, and post-pilot notes.
