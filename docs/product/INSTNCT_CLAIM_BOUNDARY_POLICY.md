# INSTNCT Claim Boundary Policy

Status: 060 license package draft.

This document is draft only. It is not final legal terms. Counsel review
required before external use. No agreement or permission starts from this draft
alone.

## Required Claim Boundary

INSTNCT materials must preserve the current claim boundary:

- research and evaluation artifacts exist
- SDK candidate exists
- local/private deployment harness exists
- public-benefit pilot boundary exists
- license package drafts exist

These do not create production, clinical, high-stakes education, public beta,
hosted SaaS, or final legal permission.

## Controlled Claims

Allowed wording:

- source-available/noncommercial draft
- commercial license requires separate written agreement
- public-benefit organizations may request review
- non-clinical only
- non-high-stakes only
- counsel review required before external use

Forbidden wording:

- production release
- public beta release
- clinical readiness
- high-stakes education readiness
- general language grounding
- consciousness
- biological/FlyWire equivalence
- physical quantum behavior
