# INSTNCT Private Pilot Success Failure Criteria

Status: 065 private pilot success/failure criteria.

These criteria determine whether a candidate may be considered ready for a
separate written approval decision. They do not approve a pilot.

## Success Criteria

- install/check smoke completes.
- deployment harness smoke completes.
- service alpha healthcheck passes.
- 063 static checker passes.
- 064 static checker passes.
- no regulated-use request appears.
- no unresolved critical issue remains.
- rollback/disable path documented.
- support owner/channel documented.
- post-pilot report template assigned to an owner.

## Failure Criteria

- regulated use appears.
- PHI/student records appear without separate written agreement.
- rollback/disable path missing.
- critical issue unresolved.
- support owner/channel missing.
- legal/counsel review required but missing.
- 061 RC_001 package check fails.
- 062 service/API alpha check fails.
- 063 security/supply-chain gate check fails.
- 064 ops readiness gate check fails.

No pilot launched.
No partner approved.
No external use authorized.
No production deployment.
No hosted SaaS.
No public beta.
No GA.
No production API readiness.
No production SRE readiness.
No SLA.
No clinical use.
No high-stakes education use.
No PHI/student records without separate written agreement.
No final legal terms.
No commercial launch.
No full VRAXION.
No language grounding.
No consciousness.

No physical quantum behavior.
