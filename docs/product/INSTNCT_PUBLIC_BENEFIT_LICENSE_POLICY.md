# INSTNCT Public-Benefit License Policy

Status: 056 productization planning artifact.

This document defines a public-benefit licensing path for approved nonprofit,
school, hospital, and research users. It is not final license text.

## Purpose

The public-benefit path allows mission-aligned organizations to use INSTNCT for
safe, bounded, noncommercial or low-cost work while preserving commercial
licensing rights.

## Eligible Organizations

Potentially eligible:

- Public schools.
- Universities.
- Public hospitals.
- Nonprofit hospitals.
- Nonprofit research labs.
- Public-interest nonprofits.
- Independent reproducibility researchers.

Eligibility is not automatic. Each use case still needs scope review.

## Allowed Public-Benefit Uses

Allowed by default only when no sensitive or high-stakes decision is delegated
to the system:

- Research reproduction.
- Classroom demonstration.
- Teacher support for non-high-stakes materials.
- Non-clinical hospital administration experiments.
- Internal document routing experiments.
- Model behavior visualization.
- Audit and evaluation of bounded research artifacts.

## Uses Requiring Separate Review

- Processing protected health information.
- Processing student education records.
- Using outputs in operational hospital or school workflows.
- Any deployment touching patient, student, or staff rights.
- Any use with minors, vulnerable populations, or emergency contexts.

## Uses Not Authorized By This Policy

- Diagnosis.
- Treatment recommendation.
- Triage.
- Medication decision.
- Clinical decision support.
- Grading.
- Admissions.
- Student ranking.
- High-stakes profiling.

056 supports productization planning only. It does not support production
