# INSTNCT Private Pilot Agreement Checklist

Status: 065 private pilot agreement checklist.

This checklist is a readiness review input only. It is not final legal terms and
does not authorize a pilot by itself.

## Required Checklist

- Candidate organization classified as non-regulated for the requested use.
- Intended use remains `local_research` or `private_evaluation`.
- Data class is allowed or separately approved in writing.
- Human owner is named.
- Support owner and support channel are named before pilot start.
- Rollback/disable owner is named.
- Incident contact is named.
- Closeout owner is named.
- Post-pilot report owner is named.
- 059 pilot boundary reviewed.
- 060 license package drafts reviewed.
- 061 RC_001 package reviewed.
- 062 service/API alpha reviewed.
- 063 security/supply-chain gate reviewed.
- 064 ops readiness gate reviewed.
- Written approval package required before any external pilot.
- Counsel review required where legal approval is needed.

## Agreement Boundary

- External support requires separate written agreement.
- External pilot requires separate written approval.
- PHI/student records require separate written agreement.
- Clinical and high-stakes education uses are outside this readiness scope.
