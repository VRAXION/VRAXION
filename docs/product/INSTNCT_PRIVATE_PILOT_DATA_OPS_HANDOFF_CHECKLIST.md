# INSTNCT Private Pilot Data Ops Handoff Checklist

Status: 065 private pilot data/ops handoff checklist.

This checklist must be complete before a candidate can move to a separate
written approval decision.

## Required Handoff Fields

- allowed data class
- forbidden data class
- retention plan
- deletion path
- audit log owner
- incident contact
- rollback/disable owner
- closeout owner
- post-pilot report owner

## Data Boundary

Allowed data class:

- synthetic data
- public non-sensitive data
- separately approved non-sensitive evaluation data

Forbidden data class:

- PHI without separate written agreement
- student records without separate written agreement
- clinical data
- grading/admissions data
- biometric data without separate written agreement
- financial data without separate written agreement
- production customer data
