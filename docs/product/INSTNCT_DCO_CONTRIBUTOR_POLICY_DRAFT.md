## Contributor Path

The default contributor path for 060 is DCO-first.

Required sign-off line:

```text
Signed-off-by: Contributor Name <contributor@example.com>
```

By submitting with a sign-off, contributor certifies right to submit
contribution and agrees the contribution can be used under project license.

CLA/copyright assignment not adopted yet. Counsel review required before
changing contributor model.

## Contributor Process Direction

- Contributors sign commits with `Signed-off-by`.
- Maintainers reject contributions without the sign-off.
- Project may later adopt a CLA only after counsel review.
- Contribution records may be retained with repository history.
