# INSTNCT Core Approved-Use Boundary

Status: 066 approved-use boundary for readiness assessment.

The only allowed future release direction described by 066 is private/self-hosted
use for approved non-regulated use. The allowed path is local/private evaluation
or controlled private deployment, and written approval is required before any
external access.

## Allowed Future Direction

```text
private/self-hosted
approved non-regulated use
local/private evaluation or controlled private deployment
written approval required
```
