# INSTNCT License Model

Status: 056 productization planning artifact.

This document defines the intended licensing posture. It is not legal advice and
is not a final license text. Final license language requires counsel review.

## License Position

INSTNCT should be described as:

```text
source-available for noncommercial research and public-interest use;
commercial use requires a separate written commercial license.
```

It should not be described as:

```text
open source but commercial use is forbidden.
```

Reason: the Open Source Initiative's Open Source Definition requires no
discrimination against fields of endeavor, including business use. A license
that restricts commercial fields is source-available/noncommercial, not OSI
open source.

## Proposed Package

1. Public source-available noncommercial license.
2. Written commercial license for paid or revenue-generating use.
3. Public-benefit rider for approved hospitals, schools, nonprofits, and
   research institutions.
4. Contributor agreement or developer certificate of origin path.
5. Trademark policy.
6. Acceptable use policy.
7. Claim boundary document.

## Commercial License Model

Commercial license terms should be separate from the public source-available
terms.

Negotiable dimensions:

- Internal evaluation.
- Internal production use.
- Hosted product integration.
- Redistribution.
- White-label use.
- Model/checkpoint sharing.
- Support and indemnity.
- Audit log retention.
- Security obligations.
- Revenue-share or royalty terms.

No default royalty rate is set in 056.

## Public-Benefit Rider

The public-benefit rider can allow no-cost or low-cost use for approved
hospitals, schools, nonprofits, and research organizations, but only inside the
allowed use boundaries.

The rider must not authorize:

- Diagnosis.
- Treatment recommendation.
- Triage.
- Medication decisions.
- Clinical decision support.
- Grading.
- Admissions.
- Student ranking.
- High-stakes profiling.
