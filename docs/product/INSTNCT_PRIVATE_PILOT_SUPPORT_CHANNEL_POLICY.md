# INSTNCT Private Pilot Support Channel Policy

Status: 065 private pilot support channel policy.

Support is best-effort evaluation support only. No production support is
created. No clinical/high-stakes support is created. No SLA is created.

## Required Support Boundary

- best-effort evaluation support only
- no SLA
- no production support
- no clinical/high-stakes support
- support channel must be named before pilot start
- support owner must be named before pilot start
- external support requires separate written agreement

## Support Intake Fields

- candidate name
- support owner
- support channel
- issue severity
- affected local/private run
- affected artifact path
- regulated-use concern flag
- rollback/disable need
- incident contact
