# INSTNCT SDK Claim Boundary

Status: 057 SDK release-candidate engineering artifact.

The SDK candidate exists to make bounded INSTNCT research workflows callable and
auditable. It does not change the product or compliance status from 056.

## Supported

- SDK release-candidate engineering.
- Structured response and error envelopes.
- Append-only progress/audit events.
- Checkpoint save/load hash verification.
- Visual export smoke through `visual_snapshot_v1`.
- Policy guard rejection for regulated use.

## Not Supported

- Production API readiness.
- Public beta.
- Clinical use.
- High-stakes education use.
- Production default training.
- Full VRAXION.
- Language grounding.
- Consciousness.
- Biological/FlyWire equivalence.
- Physical quantum behavior.
