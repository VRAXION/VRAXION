# INSTNCT Incident Response Runbook

Status: 064 local/private incident runbook draft.

This runbook is actionable for evaluation incidents only. It is not a production
support process and does not create an SLA.

## Required Roles

- incident commander
- technical lead
- communications owner
- scribe/log owner

## Required Steps

1. declare the incident and assign the incident commander.
2. triage scope, affected run, affected artifacts, and current risk.
3. contain by pausing the run, disabling the route, or blocking further use.
4. rollback/disable using the local/private runbook and checkpoint boundary.
5. communicate status to the approved internal stakeholders.
6. preserve artifacts: progress logs, audit logs, configs, summaries, and reports.
7. closeout after risk is contained and evidence is preserved.
8. postmortem using the 064 postmortem template.
