# INSTNCT Public-Benefit Rider Draft

Status: 060 license package draft.

This document is draft only. It is not final legal terms. Counsel review
required before external use. No agreement or permission starts from this draft
alone.

## Rider Direction

Eligible hospitals, schools, nonprofits, and research organizations may request
review for a public-benefit pilot candidate. Eligibility is not automatic.
Written approval required before any external pilot.

No external pilot starts from rider alone.

## Public-Benefit Restrictions

- no PHI/student records without separate written agreement
- no clinical/high-stakes use
- no diagnosis, treatment, triage, medication decisions, or clinical decision
  support
- no grading, admissions, student ranking, placement, proctoring, or
  high-stakes profiling
- no production deployment
- no hosted SaaS
- no public beta
- no customer-facing unsupervised use

## Required Pilot Conditions

- request classification completed
- data handling checklist completed
- named human owner assigned
- human oversight checklist completed
- deployment limited to `local_research` or `private_evaluation`
- rollback or disable path documented
- closeout review documented
