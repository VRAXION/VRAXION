# Legacy Harness Archive

These scripts were active one-off permutation/convergence harness entrypoints
before the `mutation_policy_ab.py` and `permutation_curve_sweeps.py`
consolidation pass.

They are preserved here as historical references only. Active maintenance and
new experiments should use:

- `v4.2/tests/mutation_policy_ab.py`
- `v4.2/tests/permutation_curve_sweeps.py`
- `v4.2/tests/harness/`
