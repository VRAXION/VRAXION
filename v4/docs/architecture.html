Ring Buffer Topology

The ring is a circular buffer of M=256 slots, each holding a D=256 vector. Six experts share this memory, reading and writing through sliding attention windows. Pointers move via golden-ratio jumps for maximal coverage.

Explorer (Je=0.9) — long phi-jumps
Walker (Jw=0.1) — short +1 steps
Neutral (0.5) — balanced blend
Attention window (2R+1 slots)

Each expert is assigned a role in round-robin: expert i gets role i % 3. With N=6: two explorers (E0, E3), two walkers (W1, W4), two neutrals (N2, N5). This guarantees both broad coverage (explorers) and local continuity (walkers) on every timestep.

Forward Pass

One call to model.forward(x) processes T timesteps sequentially. Each timestep runs N experts in series, each performing the full read-accumulate-write-move cycle.

All operations inside the expert loop are linear in v4: Linear projections + additive accumulation + scatter_add. The pointer movement uses .long() which breaks gradient flow — pointer routing is a fixed structural prior, not a learned parameter.

Soft Read / Write

The ring is accessed through a sliding attention window of 2R+1 = 5 slots centered on each expert's pointer. Reads gather and average; writes scatter-add.

Soft Read

Soft Write

Writes are additive — scatter_add accumulates information, never erases existing slot content. Weights are uniform: each of the 5 window slots gets 1/5 = 0.2 of the write value.

Pointer Movement

Each expert's pointer blends between two movement strategies:

Explorer
p = 0.9
90% phi-jump, 10% walk. Covers the ring fast — reads distant memory.
Walker
p = 0.1
10% phi-jump, 90% walk. Stays local — reinforces recent writes.
Neutral
p = 0.5
50/50 blend. Medium coverage — balanced strategy.
new_ptr = p · dest[current_slot] + (1 - p) · (current_ptr + 1) % M

The golden ratio (1/phi = 0.618...) guarantees maximally irrational spacing — no expert revisits the same slot pattern. With M=256, the jump step is floor(256 * 0.618) = 158 slots.

Gradient note: The pointer position uses .long() for slot indexing, which has zero derivative everywhere. This means the pointer path is a fixed structural prior — the network cannot learn to change WHERE it reads, only HOW it interprets what it reads (via read_proj).
